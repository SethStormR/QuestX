# QuestX
QuestX is a simple minimalistic icon theme for KDE/Gnome/XFCE/Budgie DE &amp; Cosmic DE.

Requirements
--
QuestX requires Papirus icon theme. You can get it here: https://store.kde.org/p/1166289

Tested on
--
Solus KDE/Gnome/Budgie/XFCE

Fedora KDE/Gnome/Budgie

Zorin OS 18 Pro/core/lite

PopOS! Cosmic DE
